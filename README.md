# Conexao-Banco-de-Dados-Python
Conexao Banco de Dados + Python Este é um programa básico de conexão de banco de dados(MySQL) com o Python, apresenta algumas lógicas como tratamento de erro, looping, condicional e CRUD em um banco de dados. 

# Requirements.txt
Para o programa rodar copie e cole isto no terminal, para baixar todas as bibliotecas necessárias para rodar o programa: 

pip install -r requirements.txt
